
# My Gatsby Landing Page

A Gatsby landing page similar to kontent-template-gatsby-landing-page-photon.

## 🚀 Quick start

1. **Install dependencies.**

   ```sh
   npm install
   ```

2. **Start developing.**

   ```sh
   npm run develop
   ```

3. **Open the source code and start editing!**

Your site is now running at `http://localhost:8000`!
